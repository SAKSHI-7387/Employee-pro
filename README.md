# Employee Management System

This is a web-based Employee Management System developed using Django and Python.

## Features

- Add Employee
- Update Employee
- Delete Employee
- Employee List
- Employee Search
- Login and Logout Authentication
- Dashboard

## Technologies Used

- Python
- Django
- SQL (SQLite)
- HTML
- CSS
- Bootstrap

## Installation

1. Clone the repository

git clone https://github.com/yourusername/employee-management-system.git

2. Navigate to project folder

cd employee-management-system

3. Run migrations

python manage.py makemigrations
python manage.py migrate

4. Run server

python manage.py runserver

Open in browser

http://127.0.0.1:8000
